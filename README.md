# Задание 1
## Задачи:
1. Извлечение Метода: Используйте извлечение метода для создания нового метода isOrderProcessable, который будет проверять, можно ли обработать заказ. Этот метод должен возвращать boolean значение.
2. Переименование Методов и Переменных: Измените имена методов и переменных, чтобы они точнее отражали свою функцию и были в соответствии с общепринятыми Java конвенциями.
3. Упрощение Условий: Упростите условие в if используя новый метод isOrderProcessable.
4. Опционально: Рефакторинг для Улучшения Читаемости: Проанализируйте код на предмет возможных улучшений, которые сделают его более читаемым и понятным.
